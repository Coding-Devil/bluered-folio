IoT Based Smart Parking System


Project Objectives

  • To reduce time spent manually searching for parking spots.
  • To optimize parking space usage and increase efficiencies.
  • To integrate real-time parking data with user-friendly mobile applications.

Technologies Used

  • IoT devices: Sensors and Cameras for real-time data collection.
  • Backend: Node.js, MQTT for messaging.
  • Frontend: React Native for mobile app, HTML/CSS for admin dashboard.

Features

  • Real-time availability of parking spots displayed on a mobile app.
  • Automated parking guidance system.
  • Payment integration for hassle-free transactions.
  • Analytics dashboard for monitoring and managing parking operations.
